Settings → Passwords → ⋮ (3 dots) → Export Passwords → Save CSV File

🦊 Mozilla Firefox

Settings → Privacy → Logins → ⋮ → Export Logins → Save CSV

🧭 Safari (Mac/iOS)

Safari → Settings → Passwords → ⋮ → Export All Passwords → Save CSV

🔑 Password Managers (Bitwarden / 1Password)

Settings → Tools → Export Vault → Choose CSV Format → Download

📜 Privacy Policy — Read in Detail ▼

1. Data Collection — What do we collect?

Absolutely nothing. This tool works entirely client-side (in your browser). We don't collect any personal data, passwords, usernames, URLs, or any information. There is no database, no server-side processing.

2. Data Storage — Where is your data stored?

Nowhere. Your CSV file is only loaded into your browser's temporary memory (RAM). All data is automatically erased when you refresh, close, or navigate away. No cookies, localStorage, sessionStorage, IndexedDB, or any storage mechanism is used.

3. Data Transmission — Does data go over the internet?

No. Your data is never transmitted through any server, API, third-party service, or internet connection. You can use this tool even after disconnecting the internet (once the page is loaded).

4. Third-Party Services

Only open-source libraries (jsPDF, Tailwind CSS) are used for PDF generation and styling. These libraries are also client-side and don't transmit any data. There are no analytics, ads, trackers, or third-party scripts.

5. PDF Password Protection

When you set a password on a PDF, encryption happens entirely in your browser. The password is never saved, logged, or transmitted anywhere. If you forget the password, recovering the PDF is impossible — so remember your password.

6. Open Source Transparency

This tool is a single HTML file — you can see the complete code by Right Click → View Source. There are no hidden scripts, obfuscated code, or suspicious network calls. 100% transparent and verifiable.

7. User Responsibility

Security of downloaded PDF/TXT/JSON files is your responsibility. We recommend: (a) Always add a password to PDF (b) Delete CSV file immediately (c) Keep files in encrypted storage (d) Don't share files with anyone.
